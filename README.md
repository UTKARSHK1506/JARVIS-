# ü§ñ JARVIS AI Assistant

**JARVIS** is an all-in-one AI-powered desktop assistant that can **chat, search the web in real-time, automate tasks, generate images, and speak back to you** ‚Äî all through a sleek, interactive interface. It includes features like real-time queries, automation tasks, multimedia & GUI functions, and additional utilities.

---

## üöÄ Features

### 1. General Queries
- Chat naturally with JARVIS using AI-powered responses.  
- Handles general knowledge questions with context and proper grammar.  

### 2. Real-Time Queries
- Fetch up-to-date information from the internet.  
- Provides weather updates, news, and current events dynamically.  

### 3. Automation Tasks
- Open apps like YouTube, Instagram, or system applications.  
- Search online, draft messages, play media, or automate repetitive tasks.  
- Powered by **AppOpener, pywhatkit, selenium, and keyboard**.  

### 4. Multimedia & GUI
- Generate images using AI models.  
- Speech-to-text and text-to-speech interaction using **pygame** and **edge-tts**.  
- GUI built with **PyQt5** for an interactive desktop experience.
